前后端分离大学生信息健康系统
前端采用的是Vuejs
后端技术为springboot  
数据库为mysql 版本不限制 5.7以及8.0都可以  本项目默认的为8.0
如果需要5.7的话需要在pom.xml文件中修改依赖  然后在application.properties文件中将com.mysql.cj.jdbc.Driver
修改为com.mysql.jdbc.Driver即可
本项目中涉及到三个角色   管理员学生以及老师
管理员拥有最高的权限
